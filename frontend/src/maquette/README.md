# Maquettes HTML — MarsAiFestival

Ce dossier contient les maquettes HTML statiques de toutes les pages du site.

## Règles

- Un fichier HTML par page ou section majeure
- Pas de JavaScript, pas de CSS externe — uniquement du HTML structuré
- Les classes CSS sont indicatives (elles serviront à l'IA pour comprendre le design)
- NE PAS TOUCHER — ces fichiers sont lus automatiquement par le prompt IA
